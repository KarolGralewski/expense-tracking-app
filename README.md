## Installation

### -> To run in Docker: 

1. Start docker compose in root directory with:

```
docker compose up
```

### -> To run locally: 

1. Start seperate frontend (in /client) and backend (in /server) servers with:

```
npm run dev
```

2. Start your local MongoDB server
